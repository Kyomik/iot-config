# iot-config

repi ini bertujuan untuk menyimpan script-script dari tiap alat. strukturnya sebagai berikut : [nama-alat]/[versi]/[file-versi.bin]

note:
tiap versi dari alat memiliki 2 file dengan penjelasan seperti berikut :
- version.json : terkait configurasi versi yang digunakan saat ini
- file-versi.bin : file binner dari versi
